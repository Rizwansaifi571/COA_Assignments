## Subject Focus

Computer Organization and Architecture focuses on how computer systems are designed, structured, and executed at the hardware and system level. The subject generally includes concepts such as:

- Computer system organization
- Processor architecture
- Memory hierarchy
- Instruction execution
- Input/output organization
- Number systems and data representation
- Digital logic and system-level operations
